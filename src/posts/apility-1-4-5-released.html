APIlity 1.4.5 released
Dear APIlity users,

The latest release of APIlity has just been finished. Click here to see the complete Changelog.

This release brings a minor feature enhancement and some minor bug fixes.

  • Added the possibility to select different match types for Campaign Negative Criteria. This changes the parameters and return values of the following functions:
    $campaignObject->setCampaignNegativeKeywordCriteria()
    $campaignObject->setCampaignNegativeWebsiteCriteria()
    $campaignObject->getCampaignNegativeKeywordCriteria()
    $campaignObject->getCampaignNegativeWebsiteCriteria()
    $campaignObject->toXml()
  • Fixed the getCsvReport() function which was broken in the last release.
  • Fixed a small bug in ->getAdGroupData() where an attribute was wrong.
  • Fixed a small bug in ->getCriterionData() where an attribute was missing.


Hope you like the new release.

PS: Feel free to subscribe to the APIlity announcement group.
