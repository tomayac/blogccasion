On Mac OS X with German keyboard, the '@' character is accessible via "Alt L". However, if you use Firefox and you have the LiveHTTPHeaders extension installed, "Alt L" will fire up the extension in the sidebar rather than printing '@'.

You can overcome this annoyance by disabling or remapping the hotkey for LiveHTTPHeaders. This can be easily done with the KeyConfig extension.
